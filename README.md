# IGAdBlock

Ad blocker for Instagram on a jailed iOS device

## Features

- Removes ads on reels
- Removes ads on main feed
- Removes threads suggestions on main feed

## Requirements

- Theos
- Decrypted Instagram ipa(MUST SUPPLY YOUR OWN)

## Instructions

Take the dylib from the releases, along with a dylib of a Instagram sideloading patch(to patch keychain issues) and add them to the ipa using a signing app and install on device.


## Tested On

- Instagram version 417.0 on a jailed iPhone 15 Pro Max running iOS 26.2.1
